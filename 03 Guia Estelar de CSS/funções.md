# Funções 

* nome seguido de abre e fecha parenteses
* recebe argumentos

## Exemplos

```css

@import url("http://urlaqui.com/style.css");   /* função que importa CSS */

{
    color: rgb(255,0,100);   /* função que transforma em cores RGB */

    width: calc(100% - 10%px);   /* função que calcula */
}

```
